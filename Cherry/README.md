# Cherry

Cherry ui for Opiumware @ https://discord.gg/opiumware

## Features

- **Port Attachment**: Automatically scans and attaches to Opiumware ports (8392-8397).
- **Script Execution**: Execute Lua scripts directly to the attached port.
- **Modern UI**: Sleek, dark-themed interface with Monaco Editor support.
- **Tab Management**: Dynamic tab creation and management with smart counter reset.
- **Cross-Platform**: Built with Tauri for Windows and macOS (Intel & ARM).

## Development

```powershell
# Install dependencies
npm install

# Run in dev mode
npx tauri dev
```

## Build

Built automatically via GitHub Actions for:
- Windows
- macOS (Intel & ARM)
- macOS Universal
